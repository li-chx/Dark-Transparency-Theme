# Dark Transparency Theme for VS 2026



Dark Transparency Theme for anyone who prefers dark mode. Only for version 2026.

For background please install [ClaudiaIDE](https://marketplace.visualstudio.com/items?itemName=kbuchi.ClaudiaIDE) and set "Expand to IDE" as true.



---



## About this fork



This is a fork of [Dark-Transparency-Theme](https://github.com/LITDARK/Dark-Transparency-Theme), updated for **Visual Studio 2026**.

Due to limited personal time and technical ability, this version still differs from the original. Some parts of the design may not be as polished as the original, but it is at least functional and usable.
